### Oracle 23ai Vector Database Setup

## 1. Start Oracle Container

```
docker run -d -p 1522:1521 -e ORACLE_PWD=Password123! --name oracle23ai container-registry.oracle.com/database/free:latest
```

## 2. Create User and Permissions

```
docker exec -it oracle23ai sqlplus sys/Password123!@FREEPDB1 as sysdba

CREATE USER rag_app IDENTIFIED BY password123;
GRANT DB_DEVELOPER_ROLE TO rag_app;
ALTER USER rag_app QUOTA UNLIMITED ON USERS;
EXIT
```

## 3. Create Vector Table

```
docker exec -it oracle23ai sqlplus rag_app/password123@FREEPDB1

CREATE TABLE document_chunks (
    id NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    file_name VARCHAR2(255),
    page NUMBER,
    text_content CLOB,
    raw_question CLOB,
    raw_response CLOB,
    embedding VECTOR(384, FLOAT32)
);
```

### Python Environment Setup

## 1. Create and Activate Virtual Environment

**For CPU Mode**

```
python -m venv .venv-cpu
.venv-cpu\Scripts\activate
```

**For GPU Mode**

```
python -m venv .venv-gpu
.venv-gpu\Scripts\activate
```

## 2. Install Dependencies

```
pip install -r requirements.txt
```

**CPU**

```
pip install llama-cpp-python
```

**GPU (Vulkan) - PowerShell**

```
$env:CMAKE_ARGS="-DGGML_VULKAN=on"
pip install llama-cpp-python --no-binary llama-cpp-python
```

## 3. Environment Variables

Create the .env file and set the values from the .env.example

If running side by side:

```
$env:HARDWARE_MODE="GPU"
```

## 4. Run the application

```
python -m streamlit run app.py
```

### Other commands

## Reset Database Content

```
docker exec -it oracle23ai sqlplus rag_app/password123@FREEPDB1

TRUNCATE TABLE document_chunks;
```
